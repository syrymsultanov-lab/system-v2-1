# ПРОМПТЫ ДЛЯ ГЕНЕРАЦИИ МЕДИА
# nanobanana + Veo3 (Google Gemini)
# Использовать в Claude Desktop

---

## КАК ИСПОЛЬЗОВАТЬ

1. Открой Claude Desktop
2. Создай новый чат
3. Убедись что nanobanana подключён
4. Копируй каждый промпт отдельно
5. Сохраняй результат в /landing/assets/images/ или /video/

---

## ВАЖНО ПОСЛЕ ГЕНЕРАЦИИ

1. Проверь что все 4 фото в едином стиле — тёмные, насыщенные, без людей
2. Видео должно быть плавным — подойдёт для бесконечного зацикливания
3. Если цвета слишком светлые — попроси nanobanana сделать темнее:
   ```
   Make it darker and moodier. More deep navy and emerald tones.
   Less bright. More cinematic shadow.
   ```
4. Сохраняй оригиналы — они понадобятся Claude Code
