# Design System — Premium Marine Intelligence

| Роль | Цвет | HEX |

## Стиль
- Glass-matte hybrid UI
- Editorial typography
- Асимметричный structured layout
- Subtle light refraction overlays
- Grid lines как навигационные карты
- Slow fluid motion (минимально)
