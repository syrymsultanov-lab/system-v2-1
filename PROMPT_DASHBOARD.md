Полноэкранный тёмный фон (#0B3D2E).

```
[Лого: system v2]
[Подпись: "Операционная платформа"]

[Input] Email
[Input] Пароль (с кнопкой показать/скрыть)
[Checkbox] Запомнить меня
[Button CTA] "Войти"

[Ссылка] "Забыли пароль?"
[Ссылка] "Нет доступа? → Запросить" → landing#join
```

Авторизация через Supabase Auth:
```javascript
const { data, error } = await supabase.auth.signInWithPassword({
  email, password
})
if (data.user) window.location.href = '/dashboard/'
```

При ошибке — показывать сообщение под формой.
После входа — редирект на /dashboard/.
Если уже авторизован — сразу редирект.

## ВКЛАДКА 1 — СЕГОДНЯ (tab-today)

Это главная вкладка. Открывается по умолчанию.

### Header вкладки
```
"Добрый день, [Имя]!"
"Пятница, 4 апреля 2026"  ← текущая дата
```

### Блок: Цифры дня (4 карточки в ряд)
```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│  Новые   │ │ Активных │ │  Задач   │ │ Действий │
│  лиды    │ │ контактов│ │сегодня   │ │   AI     │
│   [N]    │ │   [N]    │ │   [N]    │ │   [N]    │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```
Данные из Supabase: COUNT лидов за сегодня, COUNT активных контактов, COUNT задач, COUNT ai_jobs.

### Блок: Статус AI агента
```
┌─────────────────────────────────────┐
│  AI Агент                           │
│  ● Активен | 3 задачи выполнено     │
│  Последнее действие: 14 мин назад   │
│  [Поставить на паузу] [Подробнее]   │
└─────────────────────────────────────┘
```

### Блок: Рейтинг команды сегодня (СОРЕВНОВАТЕЛЬНЫЙ)
```
┌─────────────────────────────────────┐
│  🏆 Активность команды сегодня      │
├─────────────────────────────────────┤
│  1. Айгерим К.   ████████░░  18 дел │
│  2. Дмитрий П.   ███████░░░  15 дел │
│  3. Нурлан С.    ██████░░░░  12 дел │
│ → 4. ВЫ          █████░░░░░  10 дел │  ← выделено
│  5. Мария Т.     ████░░░░░░   8 дел │
└─────────────────────────────────────┘
* Только активность — без личных данных
```
"Дел" = сумма: обработанных лидов + закрытых задач + отправленных сообщений.
Данные из events_log за сегодня, сгруппированные по partner_id.
Имена — из таблицы partners (только первое имя + первая буква фамилии).

### Блок: Последние события (лента)
```
14:32  AI отправил сообщение → Алексей М.
```

### Блок: Быстрые действия
```

```

### Таблица лидов
```
Имя | Телефон | Мессенджер | Источник | Статус | Дата | Действия
─────────────────────────────────────────────────────────────────
Алексей М. | +7701... | WhatsApp | landing | 🟡 Новый | 04.04 | [▶][✓][✕]
Сергей К.  | +7702... | Telegram | landing | 🔵 В AI  | 03.04 | [▶][✓][✕]
```

Статусы цветами:
- 🟡 new = жёлтый
- 🔵 in_work = синий
- 🤖 handed_to_ai = фиолетовый
- 🟢 converted = зелёный
- ⛔ rejected = красный
- 📦 archived = серый

Кнопки действий на каждой строке:
- [▶] Открыть детали
- [🤖] Передать AI (меняет статус на handed_to_ai, создаёт ai_job)
- [✓] Конвертировать в контакт
- [✕] Архивировать

### Детальная карточка лида (modal или sidebar)
```
Имя: Алексей Морозов
Телефон: +77011234567
Email: alex@email.com
Мессенджер: WhatsApp (@alexmorozov)

Источник: landing
Статус: Новый
Дата регистрации: 04.04.2026 14:15

История сообщений:
[из таблицы lead_messages]

История статусов:
[из таблицы lead_status_log]

[Кнопка] "Передать AI агенту"
[Кнопка] "Конвертировать в контакт"
[Кнопка] "Архивировать"
```

## АВТОРИЗАЦИЯ И ЗАЩИТА

```javascript
// auth.js — проверять при загрузке каждой страницы
const { data: { user } } = await supabase.auth.getUser()
if (!user) window.location.href = '/login.html'

// Получить данные партнёра
const { data: partner } = await supabase
  .from('partners')
  .select('*')
  .eq('user_id', user.id)
  .single()
```

RLS в Supabase должен обеспечивать что партнёр видит ТОЛЬКО свои данные.
